## Project Overview

This project focuses on redesigning the Malawi Land Information Management System (LIMS) portal to improve usability, accessibility, and user experience. The current interface presents several usability challenges that hinder efficient land management services for citizens.

### Original Interface
- **URL:** http://lims.lands.gov.mw/
- **Purpose:** Land registration, account management, lease payments, and land information services
- **Target Users:** Malawi citizens, land professionals, government officials, businesses

### Project Goals
1. Identify and document usability problems using HCI heuristics
2. Redesign the interface following user-centered design principles
3. Create a functional prototype demonstrating improved user experience
4. Provide comprehensive evaluation and redesign justification
